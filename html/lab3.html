jQuery FAQ's

What is jQuery?

jQuery is a library of the Javascript functions that you're most likely to need as you develop web apps.

Why is jQuery so popular?

Three Reasons:

  1. It's free.
  2. It lets you get more done in less time
  3. All of it's functions are cross-browser compatible

Which is harder to learn: jQuery or Javascript?

For most functions, jQuery is significantly easier to learn and use than javascript - but remember: jQuery is javascript
